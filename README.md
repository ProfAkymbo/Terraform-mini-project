# This project creates a Load Balancer, Instances, and A Hosted Domain Name In Route 53 Using Terraform.
GOALS are to: 

### create and configure a vpc, subnet, route table, etc. with terraform
### create an 3 Ec2 instances behind a load balancer with terraform.
### create a route 53 A record and hosted domain name with terraform
### configure the 3 Ec2 instances using Ansible to output the server names with Apache
